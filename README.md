A modular TypeScript library designed to offer Web3 interfaces, enabling your
wallet to connect with decentralized applications.

```
 +----------------+            +------------------+           +---------------+
 |                |            |                  |           |               |
 |      dApps     |  <----->   |   web3 provider  |  <----->  |  your wallet  |
 |                |            |                  |           |               |
 +----------------+            +------------------+           +---------------+

```
